## [1.0.0] - 2026-01-28

### Added
- Initial release of Maintenance Cost Prediction system
- Modular ML pipeline architecture with separate components
- Data ingestion component for loading and splitting data
- Data transformation component with preprocessing pipeline
- Model trainer component with multiple algorithm evaluation
- Training pipeline for orchestrating the complete workflow
- Prediction pipeline for making predictions on new data
- Flask web application with interactive UI
- Synthetic data generator for creating realistic maintenance records
- Comprehensive Jupyter notebook with EDA and insights
- Complete documentation and README
- Setup.py for package installation
- Requirements.txt with all dependencies
- MIT License
- .gitignore for Python projects

### Features
- Support for multiple ML algorithms (Random Forest, Gradient Boosting, Linear Regression, Decision Tree)
- Automatic model selection based on performance metrics
- Real-time predictions through web interface
- Model and preprocessor persistence
- Comprehensive error handling
- Input validation
- Modular and extensible architecture

### Documentation
- Detailed README with usage instructions

- Project structure documentation

- Model performance metrics

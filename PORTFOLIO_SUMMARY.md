## üìä Project Components

### Core ML Pipeline
1. **Data Ingestion** (`src/components/data_ingestion.py`)
   - Loads raw data
   - Performs train-test split
   - Saves processed datasets

2. **Data Transformation** (`src/components/data_transformation.py`)
   - Feature engineering
   - Preprocessing pipeline
   - StandardScaler for numeric features
   - OneHotEncoder for categorical features

3. **Model Trainer** (`src/components/model_trainer.py`)
   - Multiple model evaluation
   - Performance metrics calculation
   - Best model selection
   - Model persistence

### Orchestration Pipelines
1. **Training Pipeline** (`src/pipeline/train_pipeline.py`)
   - Orchestrates complete training workflow
   - Sequential component execution
   - Progress tracking

2. **Prediction Pipeline** (`src/pipeline/predict_pipeline.py`)
   - Real-time predictions
   - Input validation
   - Model loading and inference

### Supporting Modules
- **Utils** (`src/utils.py`): Helper functions
- **Config** (`src/config.py`): Centralized configuration
- **Logger** (`src/logger.py`): Logging system
- **Exception** (`src/exception.py`): Custom exceptions

### Web Application
- **Flask App** (`app.py`): Web interface
- **Templates**: HTML UI for predictions
